# Ocode
Online Judge system

## Description

Collaborative Online Judge System is a full-stack system supporting collaborative code editing, compiling, execution and resulte judgement.

### Installing

Follow the steps to run this app.

Install libraries

```
cd oj-client
npm install

cd ../oj-server
npm install
```

Run app

```
nodemon server.js
```

Now you can use this app,
